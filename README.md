# EOSC-ORFEUS

## Irods landing page 
to retrieve/access the DO by PID [irods_landing_page](https://github.com/massimo1962/irods_landing_page).


## Irods docker
docker for irods all-in-one: icat db + iRODS [dockirods_aio](https://github.com/massimo1962/dockirods_aio).

## B2SAFE docker
